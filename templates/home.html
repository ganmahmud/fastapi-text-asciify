How to ASCIIfy

Make HTTP GET requests to /asciify and set the text parameter.

Example: /asciify?text=Hello+ASCII

Optionally you can set the font parameter to specify the output font.

Example: /asciify?text=ASCII+art&font=starwars

For available list of fonts try /fonts

To use this API in your own application try /api?text=Hello+ASCII
